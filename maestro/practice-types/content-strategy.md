# Dynamic Content Strategy

Guidelines for using web search and authentic materials in teaching.

## Core Principles

1. **Authentic materials** engage students more than artificial examples
2. **Current content** is more relevant than outdated resources
3. **Real-world context** prepares students for actual Spanish usage
4. **Level-appropriate simplification** makes authentic materials accessible

## When to Use WebSearch

### DO Search For

✅ **Real-world materials**:
- Restaurant menus: `"menú restaurante Madrid actual"`
- Store signs and advertisements
- Rental listings: `"anuncios alquiler pisos España"`
- Job postings: `"ofertas trabajo español"`

✅ **Current events**:
- News articles: `"noticias fáciles español A1 reciente"`
- Blog posts about Spanish culture
- Social media trends (described)

✅ **Regional verification**:
- `"diferencia España México ordenador computadora"`
- Pronunciation differences
- Colloquial expressions by region

✅ **Cultural context**:
- `"expresiones coloquiales México actual"`
- `"tradiciones España reciente"`
- Current popular culture references

✅ **Practical documents**:
- Forms: `"formulario alquiler España"`
- Schedules: `"horarios transporte Madrid"`
- Information signs

### DON'T Search For

❌ **Basic grammar rules**:
- Use `teaching_guides/grammar_reference.md`
- You already know conjugation patterns

❌ **Standard conjugations**:
- Verb tables are in teaching materials
- Save web search for real usage examples

❌ **Core vocabulary**:
- Use `resources/vocabulary_lists/`
- Only search for context/usage examples

❌ **General teaching principles**:
- Already in skill documentation

## Search Query Best Practices

### Time References

❌ **Avoid specific years**:
```
"menú restaurante Madrid 2025"  ← Will be outdated
"noticias España 2024"          ← Already outdated
```

✅ **Use temporal descriptors**:
```
"menú restaurante Madrid actual"
"noticias España reciente"
"información turística Barcelona"  ← No time reference needed
```

**Reason**: Avoid time-sensitive information that expires

### Difficulty Level Indicators

Add level indicators when searching for learning materials:

```
"noticias fáciles español A1"     ← For beginners
"artículos español simplificado"  ← For A2-B1
"noticias español"                ← For B2+ (no simplification needed)
```

### Regional Specification

**When region matters**:
```
"expresiones coloquiales México"    ← Mexico-specific
"menú restaurante típico España"   ← Spain-specific
"español latino vs peninsular"     ← Comparison
```

**When region doesn't matter**:
```
"verbos irregulares español"       ← Same everywhere
"gramática española"               ← Universal rules
```

## Content Processing Workflow

### Step 1: Search

Execute WebSearch with appropriate query:
```
WebSearch: [optimized query]
```

### Step 2: Assess Complexity

Determine if content matches student's CEFR level:

**Complexity indicators**:
- Vocabulary sophistication
- Sentence structure complexity
- Use of advanced tenses (subjunctive, conditional)
- Length of text
- Cultural references

### Step 3: Simplify (if needed)

**For A1 students**:
- Shorten sentences
- Replace complex vocabulary with basic equivalents
- Use only present tense
- Add translations for new words

**For A2 students**:
- Moderate simplification
- Keep most original vocabulary, explain new words
- Can include past/future tenses if taught
- Simplify complex structures

**For B1 students**:
- Minimal simplification
- Keep authentic language mostly intact
- Explain only very advanced vocabulary
- Can include subjunctive if appropriate

**For B2 students**:
- Usually no simplification needed
- Use fully authentic materials
- Explain cultural references, not language

### Step 4: Save for Reuse

```bash
# Save to appropriate category
~/spanish-learning/practice_materials/
├── news/
├── menus/
├── dialogues/
├── exercises/
├── rental_listings/
├── travel_info/
└── cultural/
```

**Naming convention**:
```
[type]-[region]-[timestamp].txt

Examples:
menu-madrid-YYYY-MM-DD.txt
news-mexico-YYYY-MM-DD.txt
rental-barcelona-YYYY-MM.txt

Note: Use actual timestamps when saving, but avoid hardcoding
specific years in documentation to prevent outdated examples.
```

### Step 5: Create Exercises

Based on authentic material, create:
- Vocabulary extraction
- Comprehension questions
- Practice scenarios
- Cultural discussion points

## Content Types & Usage

### Restaurant Menus

**Search**: `"menú restaurante [city] actual"`

**Uses**:
- Vocabulary: food items, cooking methods
- Role-play: ordering food
- Cultural: typical Spanish/Mexican dishes
- Reading comprehension

**Example lesson**:
```
[Find menu from Madrid restaurant]
[Simplify to student level]
[Extract 10 new food words]
[Role-play ordering scenario]
[Discuss Spanish vs Mexican cuisine differences]
```

### News Articles

**Search**: `"noticias fáciles español [level]"` or `"noticias [region] reciente"`

**Uses**:
- Reading comprehension
- Current vocabulary
- Discussion topics
- Cultural awareness

**Level adaptation**:
- A1: Very short articles (50-75 words), present tense only
- A2: Short articles (100-150 words), simple past included
- B1: Medium articles (200-300 words), varied tenses
- B2: Full articles (300+ words), authentic language

### Rental Listings

**Search**: `"anuncios alquiler pisos [city]"`

**Uses**:
- Housing vocabulary
- Reading authentic ads
- Writing practice (inquiry messages)
- Cultural: Spanish housing norms

**Example activities**:
- Read 3 listings, compare features
- Write inquiry message to landlord
- Discuss housing preferences
- Learn abbreviations (dorm., baño, m²)

### Travel Information

**Search**: `"información turística [city] español"` or `"guía [city]"`

**Uses**:
- Travel vocabulary
- Directions and locations
- Cultural landmarks
- Planning conversations

**Example lesson**:
```
[Find Barcelona tourism site]
[Extract info on 3 landmarks]
[Practice: "¿Cómo llego a...?" (How do I get to...?)]
[Discuss: Spanish culture and history]
```

### Job Postings

**Search**: `"ofertas trabajo español [field]"`

**Uses**: (B1-B2 levels)
- Professional vocabulary
- Formal language
- Reading job requirements
- Writing cover letters

### Social Media/Blogs

**Search**: `"blog español [topic]"` or describe social media

**Uses**: (B1-B2 levels)
- Informal language
- Current slang and expressions
- Cultural trends
- Authentic conversational Spanish

## Regional Differences Integration

### Spain vs Mexico/Latin America

**Always highlight differences** when using authentic materials:

**Vocabulary**:
```
Spain: ordenador, coche, piso, móvil
Mexico: computadora, carro, departamento, celular
```

**When teaching with real menu from Mexico**:
```
You (in user's language): "Notice they say 'elote' (corn). In Spain, they would say 'maíz'. Both are correct, just regional differences."
```

**Grammar**:
```
Spain: Vosotros tenéis (You all have)
Mexico/LA: Ustedes tienen (You all have)
```

**Point out in authentic text**:
```
[If Spanish website uses "vosotros"]
You (in user's language): "See this 'vosotros' form? That's used in Spain. In Latin America, they use 'ustedes' instead. Both mean 'you all', but different regions prefer different forms."
```

**Cultural context**:
- Food differences (paella vs tacos)
- Customs and traditions
- Historical context
- Current events by region

Reference: `teaching_guides/regional_differences.md`

## Quality Criteria

Good authentic materials are:
- ✅ Actually from Spanish-language sources
- ✅ Current and relevant
- ✅ Culturally authentic
- ✅ Appropriate level (or simplified appropriately)
- ✅ Connected to student interests/goals
- ✅ Useful for practice and engagement

Poor materials are:
- ❌ Machine-translated or artificial
- ❌ Outdated
- ❌ Too complex with no simplification path
- ❌ Irrelevant to student
- ❌ Culturally inappropriate

## Integration with Curriculum

**Structured learning** (Type 1):
- Use authentic materials to supplement curriculum topics
- Example: Week teaching food vocabulary → find real menus

**Quick practice** (Type 2):
- Student-driven authentic material requests
- More flexible, interest-based content

**Assessment** (Type 3):
- Use authentic materials for reading comprehension tests
- Cultural questions based on real content

## Success Metrics

Effective use of authentic materials shows:
- ✅ Student engagement increases
- ✅ Student sees real-world application
- ✅ Vocabulary becomes more practical
- ✅ Cultural understanding deepens
- ✅ Student confidence with real Spanish grows
