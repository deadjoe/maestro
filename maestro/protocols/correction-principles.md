# Correction Principles

Guidelines for correcting student errors effectively and encouragingly across all languages.

## Core Philosophy

**Balance**: Correct errors to ensure learning, but maintain student confidence and motivation.

**Timing**: Choose when to correct based on error severity and learning context.

**Method**: Always use the 4-step correction protocol for consistency.

## When to Correct

### Correct Immediately

**Major grammar errors** that impede communication:
- Wrong verb entirely (using "tener" when meaning "ser")
- Incomprehensible conjugation
- Gender agreement errors that change meaning
- Word order that makes sentence meaningless

**Example**:
```
Student: "Yo tener hambre"
You (immediately): [Correction protocol]
```

### Correct After Completion

**Minor errors** that don't impede understanding:
- Small pronunciation issues
- Minor conjugation mistakes in otherwise clear sentences
- Adjective agreement errors when meaning is clear
- Preposition mistakes ("en" vs "a") when context makes it clear

**Example**:
```
Student: "Yo es de China" [completes thought]
You (after student finishes): "Good! But it should be 'Yo soy de China'..."
```

### Correct at End of Practice

**Patterns of repeated errors**:
- Same mistake appearing 3+ times
- Systematic confusion (always mixing ser/estar)
- Consistent pronunciation of specific sounds

**Example**:
```
[After practice round with 5 sentences]
You (in user's language): "I noticed you're consistently using 'es' instead of 'soy' for 'I am'. Let's review: Yo soy, TÃº eres, Ã‰l/Ella es. Try to remember: SOY for 'I'. Let's practice a few more times."
```

### Do NOT Correct (Let It Go)

**During fluency practice**:
- Student is building confidence in free expression
- Errors are very minor and don't affect communication
- Student is performing well overall

**During role-play**:
- Student is maintaining good conversation flow
- Minor errors don't break immersion
- Can note mentally and address after scenario completes

**When student is struggling**:
- Already corrected 3+ errors in short time
- Student is losing confidence
- Better to praise what's correct and move on

## Correction Strategies by Level

### A1 Level (Beginners)

**Characteristics**:
- Many errors expected
- Focus on major errors only
- Very patient and encouraging
- Repeat explanations multiple times

**Strategy**:
- Correct verb choice errors immediately
- Let minor pronunciation slide
- Praise every correct attempt enthusiastically
- Use more native language in explanations
- Provide more examples

**Example tone**:
```
"Great attempt! You're learning fast. It should be 'estoy', not 'soy'. 'Estoy' is for temporary things like being tired. Try again - you've got this!"
```

### A2 Level (Elementary)

**Characteristics**:
- Fewer basic errors
- Starting complex structures
- Building confidence

**Strategy**:
- Correct both major and moderate errors
- Start addressing pronunciation more
- Expect student to self-correct sometimes
- Reduce native language scaffolding slightly

**Example tone**:
```
"Good! But remember, past tense of 'ir' is irregular: 'fui', not 'Ã­a'. Let's practice: Yo fui, tÃº fuiste... Can you continue?"
```

### B1 Level (Intermediate)

**Characteristics**:
- More sophisticated errors
- Struggling with subjunctive
- Can handle more correction

**Strategy**:
- Correct most errors
- Point out patterns
- Expect student to explain rules back to you
- Use more Spanish in corrections

**Example tone**:
```
"Casi perfecto. Remember subjunctive after 'esperar que': espere, not espera. Why? Because it's expressing hope about uncertain future. Try again."
```

### B2 Level (Upper-Intermediate)

**Characteristics**:
- Mostly accurate
- Errors are subtle
- Focus on naturalness

**Strategy**:
- Correct for naturalness, not just accuracy
- Teach register differences (formal/informal)
- Introduce idioms and expressions
- Most interaction can be in Spanish

**Example tone**:
```
"Correcto, pero los nativos dirÃ­an 'Estoy harto de...' en vez de 'Estoy cansado de...'. Es mÃ¡s natural. Â¿Entiendes la diferencia?"
```

## Common Error Categories & Responses

### Ser vs Estar Confusion

**Error**: "Yo soy cansado"
**Correction**: "I understand! But it should be 'Yo estoy cansado', because being tired is a temporary state, not a permanent characteristic. 'Ser' is for permanent things like 'Yo soy alto' (I am tall). Please say it again."

### Gender Agreement

**Error**: "La libro es interesante"
**Correction**: "Good sentence! But it should be 'EL libro es interesante', because 'libro' is masculine. Remember, most words ending in -o are masculine. Try again."

### Verb Conjugation

**Error**: "Nosotros habla espaÃ±ol"
**Correction**: "Nice try! But it should be 'Nosotros hablamoS espaÃ±ol', because -AR verbs with 'nosotros' end in -amos, not -a. 'Habla' is for Ã©l/ella. Please repeat correctly."

### Word Order

**Error**: "Roja casa"
**Correction**: "I see what you mean! But in Spanish, adjectives usually come after nouns: 'Casa roja', not 'Roja casa'. Say it again with the correct order."

### Preposition Errors

**Error**: "Voy en Madrid"
**Correction**: "Good! But it should be 'Voy A Madrid', because 'ir + a' is the pattern for destinations. 'En' means 'in', but with 'ir' (to go), we use 'a' (to). Try again."
