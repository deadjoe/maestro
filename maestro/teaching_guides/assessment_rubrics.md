# Assessment Rubrics - Evaluation Standards

## Overview

This document provides detailed scoring criteria for assessing student performance across all CEFR levels (A1-B2). Every weekly assessment (Day 5) uses these rubrics.

---

## 4-Dimension Evaluation Framework

All assessments score students on **4 dimensions**, each worth **25 points**, for a total of **100 points**.

### Dimension 1: Grammar Accuracy (25 points)
### Dimension 2: Vocabulary Range (25 points)
### Dimension 3: Fluency & Communication (25 points)
### Dimension 4: Comprehension (25 points)

---

## Dimension 1: Grammar Accuracy (25 points)

**What we evaluate**:
- Verb conjugation correctness
- Gender and number agreement
- Sentence structure
- Appropriate tense usage
- Subjunctive usage (B1+)

### A1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Consistently correct basic conjugations (ser, estar, regular present). Rare gender/number errors. Can form simple questions correctly. |
| 20-22 | Mostly correct present tense. Occasional errors with ser/estar or irregular verbs. Generally correct gender agreement. |
| 17-19 | Frequent minor errors but communicates meaning. Confusion with ser/estar. Some gender agreement issues. |
| 14-16 | Many errors affecting clarity. Struggles with verb conjugation. Frequent gender errors. |
| 0-13  | Unable to form correct basic sentences. Major conjugation and agreement issues. |

### A2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Correct use of preterite and imperfect in appropriate contexts. Good control of present and near future. Minor errors only. |
| 20-22 | Generally correct past tenses with occasional confusion. Good present tense control. Some errors with irregular preterites. |
| 17-19 | Can use past tenses but with frequent errors. Preterite/imperfect distinction unclear. Present tense mostly correct. |
| 14-16 | Struggles with past tenses. Many errors but basic meaning clear. |
| 0-13  | Cannot reliably use past tenses. Major structural errors. |

### B1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Correct subjunctive usage in most contexts (WEIRDO). Good conditional and si clauses. All tenses controlled. Minor errors only. |
| 20-22 | Generally correct subjunctive with common triggers. Good conditional usage. Occasional errors with imperfect subjunctive. |
| 17-19 | Can use subjunctive but with errors. Understands concept but struggles with consistent application. Other tenses good. |
| 14-16 | Subjunctive usage inconsistent. Errors with conditional. Other tenses mostly controlled. |
| 0-13  | Cannot use subjunctive appropriately. Major errors with complex structures. |

### B2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Sophisticated, accurate use of all moods and tenses. Subjunctive in all clause types. Occasional minor slips only. Native-like accuracy. |
| 20-22 | Strong control of subjunctive and conditional. All tenses accurate. Minor errors in complex structures. |
| 17-19 | Good command of grammar. Some errors in advanced subjunctive contexts (adjective clauses, adverbial). |
| 14-16 | Adequate grammar but errors increase with complexity. Can use advanced structures with effort. |
| 0-13  | Frequent errors affecting meaning. Cannot sustain advanced structures. |

## Dimension 2: Vocabulary Range (25 points)

**What we evaluate**:
- Breadth of vocabulary
- Appropriateness to context
- Level-appropriate usage
- Avoidance of repetition
- Idiomatic expressions (B2)

### A1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Uses 40-50 level-appropriate words. Varied expressions. Minimal repetition. Covers week's vocabulary well. |
| 20-22 | Uses 30-40 words. Some variety. Occasional repetition. Most key vocabulary included. |
| 17-19 | Uses 20-30 words. Frequent repetition. Limited range but covers basic needs. |
| 14-16 | Uses 15-25 words. Heavy repetition. Basic vocabulary only. |
| 0-13  | Very limited vocabulary (<15 words). Cannot express basic ideas adequately. |

### A2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Rich vocabulary for level (60-80 words). Past tense vocabulary. Descriptive language. Good variety. |
| 20-22 | Adequate range (50-70 words). Can describe past and future. Some descriptive adjectives. |
| 17-19 | Functional vocabulary (40-60 words). Basic descriptions. Some repetition. |
| 14-16 | Limited range (30-50 words). Struggles with descriptive language. |
| 0-13  | Insufficient vocabulary (<30 words). Cannot describe beyond basic present. |

### B1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Wide range (100+ words). Abstract concepts. Opinion vocabulary. Discourse markers. Sophisticated expression. |
| 20-22 | Good range (80-100 words). Can express opinions and hypotheticals. Uses connectors. Some abstraction. |
| 17-19 | Adequate range (60-80 words). Limited abstract vocabulary. Basic connectors. Some paraphrasing needed. |
| 14-16 | Limited range (40-60 words). Struggles with abstract concepts. Frequent paraphrasing. |
| 0-13  | Insufficient range (<40 words). Cannot express complex ideas. Heavy repetition. |

### B2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Very wide range (150+ words). Idiomatic expressions. Precise word choice. Register-appropriate. Nuanced vocabulary. |
| 20-22 | Wide range (120-150 words). Some idioms. Generally precise. Good colloquialisms. |
| 17-19 | Good range (100-120 words). Occasional idioms. Mostly appropriate. Some imprecision. |
| 14-16 | Adequate range (80-100 words). Few idioms. Some inappropriate choices. |
| 0-13  | Limited range (<80 words). No idiomatic usage. Frequent imprecision. |

## Dimension 3: Fluency & Communication (25 points)

**What we evaluate**:
- Speaking pace and hesitation
- Self-correction ability
- Communication effectiveness
- Interaction skills
- Natural flow

### A1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Speaks with minimal hesitation on familiar topics. Can ask clarification. Self-corrects some errors. Communicates effectively. |
| 20-22 | Some hesitation but maintains communication. Occasionally asks for help. Completes basic exchanges. |
| 17-19 | Frequent pauses. Requires encouragement. Completes tasks with support. Communication sometimes breaks down. |
| 14-16 | Very slow. Long pauses. Needs significant support. Often cannot complete ideas. |
| 0-13  | Cannot maintain basic communication. Constant struggle. Rarely completes exchanges. |

### A2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Generally fluent on familiar topics. Can narrate past events. Handles unexpected questions reasonably. Good interaction. |
| 20-22 | Adequate fluency. Some hesitation with past tenses. Handles most familiar situations. |
| 17-19 | Noticeable hesitation. Struggles with narrative flow. Can communicate but slowly. |
| 14-16 | Frequent breakdowns. Difficulty maintaining flow. Communication effortful. |
| 0-13  | Cannot maintain communication beyond simple exchanges. Very effortful. |

### B1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Fluent and spontaneous. Can discuss abstract topics. Handles unexpected questions well. Natural interaction. Good linking of ideas. |
| 20-22 | Generally fluent. Occasional hesitation with complex topics. Good spontaneity. Can maintain discussion. |
| 17-19 | Adequate fluency on familiar topics. Hesitation with abstract concepts. Can maintain interaction with effort. |
| 14-16 | Frequent hesitation. Struggles with spontaneity. Interaction often effortful. |
| 0-13  | Cannot maintain fluent communication. Constant hesitation. Poor interaction. |

### B2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Very fluent and natural. Native-like interaction. Handles all topics spontaneously. Sophisticated linking. Minimal hesitation. |
| 20-22 | Fluent and spontaneous. Can discuss any topic. Occasional hesitation with very complex ideas. Good interaction. |
| 17-19 | Generally fluent. Some hesitation with sophisticated concepts. Can maintain complex discussions. |
| 14-16 | Adequate fluency but increases effort with complexity. Noticeable hesitation. Can communicate but not smoothly. |
| 0-13  | Cannot maintain fluent communication at B2 level. Frequent breakdowns. |

## Dimension 4: Comprehension (25 points)

**What we evaluate**:
- Understanding questions and prompts
- Responding relevantly
- Following instructions
- Asking for clarification appropriately
- Understanding context

### A1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Understands all simple questions and instructions. Responds appropriately. Rarely needs repetition. |
| 20-22 | Understands most simple questions. Occasionally asks for repetition. Generally responds correctly. |
| 17-19 | Understands with some difficulty. Frequently needs repetition or clarification. Sometimes misunderstands. |
| 14-16 | Struggles to understand. Often misunderstands questions. Needs significant support. |
| 0-13  | Cannot understand basic questions consistently. Frequent misunderstandings. |

### A2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Understands detailed questions about familiar topics. Follows narratives. Responds relevantly. |
| 20-22 | Understands most questions. Can follow simple narratives. Generally responds appropriately. |
| 17-19 | Understands main points. Some details missed. Occasional irrelevant responses. |
| 14-16 | Struggles with detailed questions. Misses many details. Often needs clarification. |
| 0-13  | Cannot understand beyond basic questions. Frequent misunderstandings. |

### B1 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Understands complex questions and abstract discussions. Grasps nuances. Always responds relevantly. |
| 20-22 | Understands most complex input. Occasional misses on subtleties. Generally excellent comprehension. |
| 17-19 | Understands main ideas of complex input. Some nuances missed. Mostly relevant responses. |
| 14-16 | Struggles with complex or abstract input. Misses nuances frequently. Sometimes off-topic. |
| 0-13  | Cannot understand complex input. Frequent misunderstandings. Often irrelevant responses. |

### B2 Level

| Score | Criteria |
|-------|----------|
| 23-25 | Understands virtually everything, including implicit meanings, humor, and irony. Native-like comprehension. |
| 20-22 | Excellent comprehension. Grasps most implicit meanings. Rare misunderstandings. |
| 17-19 | Very good comprehension. Occasionally misses subtle implications. Understands most humor and idioms. |
| 14-16 | Good comprehension of explicit meaning. Struggles with implicit meanings and cultural references. |
| 0-13  | Cannot reliably understand B2-level input. Misses implicit meanings. |

## Level Advancement Criteria

### A1 → A2 Advancement
**Requirements**:
- Complete Week 12 assessment
- Score ≥80/100 on final A1 comprehensive assessment
- Demonstrate consistent grammar in present tense
- Vocabulary of ~500 words actively used
- Can handle basic daily interactions independently

**If score 70-79**: Review weak areas, then advance
**If score <70**: Repeat selected A1 weeks focusing on weaknesses

### A2 → B1 Advancement
**Requirements**:
- Complete Week 24 assessment
- Score ≥80/100 on final A2 comprehensive assessment
- Master preterite and imperfect tenses
- Comfortable with future tenses
- Vocabulary of ~1000 words
- Can narrate past experiences fluently

**If score 70-79**: Review past tenses, then advance
**If score <70**: Focus on past tense mastery before advancing

### B1 → B2 Advancement
**Requirements**:
- Complete Week 40 assessment
- Score ≥80/100 on final B1 comprehensive assessment
- **3 consecutive weekly assessments ≥80/100** (Weeks 38, 39, 40)
- Subjunctive used accurately in most contexts
- Vocabulary of ~2000 words
- Can express and justify opinions on complex topics

**If score 70-79**: Review subjunctive mastery, then advance
**If score <70**: Extended B1 practice needed

### B2 Completion
**Requirements**:
- Complete Week 52 assessment
- Score ≥80/100 on comprehensive B2 assessment
- **3 consecutive weekly assessments ≥80/100** (Weeks 50, 51, 52)
- Fluent, spontaneous communication
- Sophisticated grammar usage
- Vocabulary of ~4000+ words
- Can function independently in Spanish-speaking environment

**B2 achieved**: Celebrate! Student is now an independent user of Spanish.

## Assessment Best Practices

### During Assessment
1. **Create comfortable environment**: Reduce stress, encourage natural performance
2. **Be consistent**: Use same rubrics for all students at same level
3. **Take notes**: Record specific examples during speaking assessment
4. **Focus on communication**: Don't interrupt excessively to correct
5. **Holistic view**: Consider progress, not just current performance

### After Assessment
1. **Provide detailed feedback**: Use template above
2. **Be specific**: Give examples of errors and corrections
3. **Encourage**: Highlight achievements, not just weaknesses
4. **Set clear goals**: What to focus on next week
5. **Update progress.md**: Record results and plan
6. **Update vocab_tracker**: Add words needing review

### Weekly vs Level Assessments
**Weekly assessments** (Day 5):
- Focus on current week's content
- Identify immediate needs
- Guide next week's focus
- Less formal, more diagnostic

**Level assessments** (Week 12, 24, 40, 52):
- Comprehensive evaluation of all level content
- Determine level advancement
- More formal and thorough
- Include all skills (speaking, writing, listening, reading)

## Common Assessment Challenges

### Challenge 1: Student Nervousness
**Solution**:
- Start with easier warm-up questions
- Remind that mistakes are learning opportunities
- Focus on what they CAN do, not just errors
- Create conversational atmosphere, not exam feeling

### Challenge 2: Inconsistent Performance
**Solution**:
- Consider multiple data points, not just one assessment
- Account for off days
- Focus on general trend, not single score
- Ask student about factors affecting performance

### Challenge 3: Scoring Borderline Cases
**Solution**:
- When between two scores, consider overall communication effectiveness
- Trend matters: improving or declining?
- Give benefit of doubt for students showing clear effort
- Be consistent with your own past scoring

### Challenge 4: Student Disagrees with Score
**Solution**:
- Show specific examples supporting your evaluation
- Refer to rubric criteria
- Acknowledge their perspective
- Focus discussion on improvement plan, not just the number

## Recording and Tracking

### After Each Assessment
1. **Save results** to `~/spanish-learning/weekly_reports/week_XX_assessment.md`
2. **Update progress.md** with scores and progress notes
3. **Run progress analyzer**:
   ```bash
   python maestro/scripts/progress_analyzer.py --record-assessment [score] [week]
   ```
4. **Generate report** (if end of level):
   ```bash
   python maestro/scripts/progress_analyzer.py --generate-report level
   ```

### Track Trends
- Weekly scores over time
- Dimension-specific strengths/weaknesses
- Vocabulary growth
- Fluency improvements
- Comprehension development

Use `progress_analyzer.py` to visualize these trends and identify patterns.
